CALCULADORA DE IMC

 Ejecutar el servidor:
   python manage.py runserver

 Abrir en el navegador:
   http://127.0.0.1:8000/

Funcionalidades:

- Ingrese sus datos: peso (kg) y altura (m)
- Haga clic en "Calcular IMC"
- Vea su resultado y categoria
- Use "Limpiar campos" para nuevos datos

Clasificacion de IMC:
- Menor a 18.5: Bajo peso
- 18.5 - 24.9: Peso normal
- 25 - 29.9: Sobrepeso
- 30 o mas: Obesidad
